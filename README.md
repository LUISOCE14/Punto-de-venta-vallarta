# Punto-de-venta-vallarta

## Aplicacion para llevar la gestion de los inventarios  de un empresa que vende muebles y las ventas
App con ventas para registrar las nuevas ventas que se hacen dentro de la tienda y un modulo para registrar nuevos clientes, el inventario es para registrar compras que a hecho la empresa para restaurar el stock
y tambien para cambiar o a√±adir nuevos productos que vallan llegando, ademas tiene un modulo de administracion que puede acceder a todos los modulos ademas de registrar nuevos empleados y asignar un rol segun el nivel que tenga.Ademas el sistema tiene un admistrador de roles y cuando inicia sesion solo se muestra los menus disponibles para los diferentes roles
